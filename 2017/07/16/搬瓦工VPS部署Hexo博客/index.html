配置免密登陆VPS

在Git bash中执行
ssh-keygen -t rsa -C "email@example.com

默认设置，一路回车，你就会发现~/.ssh目录下多出了两个文件一个私钥文件id_rsa，一个公钥文件id_rsa.pub。我们只要将本地的公钥复制到VPS主机的~/.ssh目录下的~/.ssh/authorized_keys文件中就能免密登陆服务器。

配置好后用ssh git@67.209.***.***就能登陆上VPS
